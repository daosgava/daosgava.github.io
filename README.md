# Personal Website

This is a static website to showcase my professional profile, experience, and recent projects.

## Technologies Used
- Bun
- Vite
- TypeScript
- HTML
- CSS
- CI/CD with GitHub Actions

## Getting Started

### Run in Development Mode
```sh
bun dev
```

### Build for Production
```sh
bun run build
```

### Preview the Build
```sh
bun run preview
```
